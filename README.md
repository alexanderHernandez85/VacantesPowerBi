# VacantesPowerBi
Este proyecto analiza el impacto de las IA generativas en el mercado laboral de analistas de datos entre 2024 y 2025. Los siguientes datos son una muestra de empleos dentro del área del análisis de datos. Dichos datos están organizados por cargo, salario, experiencia, tipo de empleo, país de la compañía, habilidades requeridas, etc...
